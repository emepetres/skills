# skills

Practical skills for my everyday needs.

## Quickstart

Install these skills with the `skills` installer:

```bash
npx skills@latest add emepetres/skills
```

Then:

1. Pick the skills you want to install.
2. Choose which coding agent(s) you want to install them for.
3. Restart your agent if needed so it picks up the new skills.

## Available skills

Current installable skills in this repo:

- `faithful-translate`

## Manual install

If you prefer, you can also copy a skill directory from `skills/` into your agent's local skills directory and reload the agent.
